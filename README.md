# wp1101期末專題 - Bomb It

## 安裝步驟

```bash
git clone [link]
cd WPFinalBombit
cd bombit && yarn
cd ../backend && yarn
# 新增 .env 檔並填入 MONGO_URL
```



## 測試步驟

```bash
# frontend
cd bombit && yarn start

# backend
cd backend && yarn server
```
